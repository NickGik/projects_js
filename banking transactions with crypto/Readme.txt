Baking
Добро пожаловать в Baking - фронтэнд часть нашего амбициозного проекта крипто-банка!

О проекте
Baking - это фронтэнд приложение, которое будет использоваться для создания интуитивно понятного и удобного 
интерфейса для крипто-банка. Наша миссия - сделать банковские операции с криптовалютами доступными и простыми 
для всех.

Технологии
React: библиотека для создания пользовательских интерфейсов
React Router: библиотека для управления навигацией
React Query: библиотека для управления данными
Chart.js: библиотека для создания диаграмм
Vite: инструмент для быстрой разработки и сборки
Cypress: библиотека для тестирования
Jest: фреймворк для тестирования
ESLint: инструмент для проверки кода
Sass: препроцессор для CSS
Настройка проекта

Установка:
git clone [репозиторий]
cd baking
npm install

Запуск разработки:
npm run dev

Это запустит сервер разработки, который автоматически перезагрузится при изменении файлов.
Сборка:
npm run build

Это создаст оптимизированную версию приложения в директории dist.
Тестирование:
npm run test

Это запустит тесты с помощью Jest.
Запуск тестов в Cypress:
npm run cypress:open
Это откроет интерфейс Cypress для запуска и просмотра тестов.
