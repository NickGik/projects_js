CRM-система для управления клиентами

Обзор проекта
Данный проект представляет собой CRM-систему для управления клиентской базой. Система
написана на JavaScript и состоит из двух частей:
Backend (crm-backend): Отвечает за хранение и обработку данных о клиентах.
Frontend: Отвечает за визуальное представление данных и взаимодействие с пользователем.

Установка и запуск
Backend (crm-backend)
Клонирование репозитория:
cd crm-backend

Установка зависимостей:
npm install

Запуск сервера:
npm start

Frontend
Запуск приложения:
npm start
